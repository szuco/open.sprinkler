# Summary

This is a case for the TIAO OpenSprinkler Pi sprinkler controller.  The sprinkler box is meant to be weather proof and originally had a spot for a o-ring, but instead I just silicon the lid.  There is a right-angle pipe coming in that you run the wires in to the box.  I screw mine down to the top of the valve box cover and water stays out.
